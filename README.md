### Webworker

Webworker is a Large Language Model (LLMs) based web agent that can complete user instructions end-to-end by interacting with real-world websites.

- Webworker that integrates textual and visual information to address web tasks end-to-end and introduce a generalist planning approach for navigation.
- Webworker that build an online web browsing environment using Selenium.

#### set up

- conda create -n webworker python=3.10
- conda activate webworker

#### run

- bash run.sh
